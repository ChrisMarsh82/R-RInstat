Description

Gets an S3 method.

Usage

## Default S3 method:
getMethodS3(name, class="default", envir=parent.frame(), ...)

Arguments

name

The name of the method.

class

The class of the method.

envir

The environment from which the search for the S3 method is done.

...

Not used.
